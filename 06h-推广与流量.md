# 推广与流量

> 所属：[06 运营方案](06-运营方案.md) > 第三部分：运营手册

## 推广前置条件

**在售商品数量达到 300 款之前，不开启任何付费推广。** 推广预算在商品池未达规模时投入产出比极低，应将精力集中在选品上架上。

## 直通车推广策略（低价引流模式）

### 核心参数

| 参数 | 设置 |
|------|------|
| **推广类型** | 关键词点击推广（标准计划） |
| **单次点击出价上限** | **≤ ￥0.20** |
| **日消耗上限** | **￥30/天** |
| **开启时机** | 在售商品满 **300 款** 后批量开启 |
| **辅助工具** | 省心快车插件（由老板开通后启用，可自动优化出价与匹配） |

### 开车流程

| 步骤 | 执行者 | 动作 |
|------|--------|------|
| 1 | 聚宝 | 确认在售商品 ≥ 300 款，发起「开车指令」 |
| 2 | 执远 | 批量为所有在售商品创建直通车关键词点击推广计划 |
| 3 | 执远 | 设置统一出价 ≤ ￥0.20，日限额 ￥30 |
| 4 | 执远 | 若省心快车插件已开通，启用自动优化模式 |
| 5 | 聚宝 | 推广开启后，继续并行选品上架（不因推广而暂停上新） |

### 推广效果清理（按周节奏）

推广开启后，按以下时间窗口逐级淘汰无效关键词/商品计划：

| 时间节点 | 淘汰条件 | 执行者 | 动作 |
|----------|----------|--------|------|
| **第 1 周末** | 关键词 **无展现量** | 执远 | 删除该关键词，换词或暂停该商品计划 |
| **第 2 周末** | 关键词有展现但 **无点击** | 执远 | 删除该关键词（展现有但没人点，说明创意或匹配有问题） |
| **第 3 周末** | 关键词有点击但 **无成交** | 执远 | 删除该关键词（有流量但不转化，亏钱词） |

**清理后的补充** ：
- 被清理的商品若仍在售，聚宝评估是否值得换一批关键词重新开车
- 3 周内有成交的关键词保留并持续观察 ROI

### 推广循环

推广与选品上架并行进行，形成持续循环：

```text
选品上架 → 攒满 300 款 → 批量开车 → 按周清理无效词
     ↑                                    ↓
     └──── 持续并行选品上架 ←──── 满 300 新品再开新一轮计划
```

每轮推广计划以当前所有在售商品为池，新上架的商品在下一个"满 300"节点纳入新计划。

## 免费流量

| 渠道 | 聚宝职责 | 协作方 |
|------|----------|--------|
| **搜索 SEO** | 确定核心关键词策略 → 下工单 | 美含（标题优化、属性填写） |
| **内容种草** | 确定内容方向与选题 → 下工单 | 美含（图文/短视频制作） |
| **微淘/逛逛** | 制定内容日历 → 下工单 | 美含（内容生产） |

```text

- 直通车: 花费 ￥{XXX} / ROI {X.X} / PPC ￥{X.XX} / 点击率 {X.X%}
- 在售商品数: {XXX} | 推广中商品数: {XXX}

- 异常/调整: {无或简述}
```
