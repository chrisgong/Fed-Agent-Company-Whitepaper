# 推广与流量（蓝海词专项优化版）

> 所属：[06 运营方案](06-运营方案.md) > 第三部分：运营手册

**核心策略**：「蓝海词店铺」—— 寻找高搜索、低竞争、精准需求的细分市场。推广方案的核心是**「低成本探测」**与**「精准收割」**，而非烧钱砸流量。

蓝海词模式要点：**靠量铺货探测，靠词获取精准流，靠极低 PPC（点击单价）保住利润。**

---

## 一、推广前置条件：规模化探测

**核心逻辑**：蓝海店是「概率游戏」。

* **在售商品数 < 300 款**：执行「全店无感期」策略。**禁止开启付费推广**，所有精力由 **【聚宝】** 牵头，**【如一】** 审计，完成高蓝海指数词的选品与标题上架。
* **蓝海指数要求**：所有上架商品的标题核心词，其「搜索意向/在线商品数」比例必须 **> 2**（由 **【如一】** 在上架前进行数据校验）。

---

## 二、直通车推广策略：低价「钓鱼」模式

蓝海计划不抢首屏，要的是**全网捡漏**。

### 1. 核心参数优化

| 参数 | 设置 | 专家建议 |
| :--- | :--- | :--- |
| **推广类型** | 关键词标准计划 | 采用「多计划、多商品、极低出价」的钓鱼模式。 |
| **点击出价上限** | **￥0.10 - ￥0.30** | 根据类目微调。蓝海词竞争小，0.2 元通常能排在后排或侧边，正是精准低价流的位置。 |
| **日消耗上限** | **￥30 - ￥50/天** | 初始设置 30，若 ROI > 2 且预算下线早，由【聚宝】申请【发财】提额。 |
| **匹配方式** | **精准匹配 / 广泛匹配** | 核心蓝海词用精准，长尾延伸词用广泛。 |

### 2. 「开车」流程自动化

| 步骤 | 执行者 | 动作逻辑 |
| :--- | :--- | :--- |
| 1 | **聚宝** | 确认在售商品 ≥ 300，从【如一】处获取「高潜力商品清单」（收藏加购比 > 5% 的款）。 |
| 2 | **执远** | 批量创建计划。每个商品挂载 5-10 个精准蓝海长尾词，出价统一 0.2 元。 |
| 3 | **如一** | 监控 Token 消耗与直通车消耗比例，确保「推广费 < 毛利的 20%」。 |
| 4 | **美含** | 针对推广计划中的 Top 10 商品，二次优化「高点击车图」（突出蓝海需求痛点）。 |

## 三、推广动态清理：三级淘汰制（按周）

蓝海店最怕「僵尸计划」占用预算。将「成交」指标前置，并增加「潜力」指标。

| 时间节点 | 淘汰/优化条件 | 执行者 | 动作说明 |
| :--- | :--- | :--- | :--- |
| **第 1 周末** | **无展现 / 展现 < 50** | **执远** | 说明词太偏或权重不足。动作：提升出价 0.05 元测试，若持续无展现则删词。 |
| **第 2 周末** | **点击率 < 行业均值** | **美含** | 说明图不行或词不准。动作：修改主图文案，或由【聚宝】更换核心词。 |
| **第 3 周末** | **无收藏加购 & 无成交** | **执远** | 判定为「无效流量」。动作：**彻底删除该商品计划**，腾出预算给新品。 |

**专家补丁**：

* **潜力款锁定**：若 3 周内虽无成交，但**收藏加购率 > 8%**，不删除，交由【美含】优化详情页文案，交由【聚宝】制定促销（如发券）刺激转化。

## 四、免费流量：SEO 与内容权重

| 渠道 | 核心策略 | 协作细节 |
| :--- | :--- | :--- |
| **搜索 SEO** | **卡位蓝海长尾词** | **【如一】** 提供高搜索量/低竞争词表 → **【美含】** 编写「三段式」标题。 |
| **逛逛/种草** | **解决信任感** | **【美含】** 针对蓝海产品特有的「奇葩」或「专业」需求编写评测文案。 |
| **自然权重** | **动销率维护** | **【执远】** 定期清理 30 天无流量滞销品，保持全店权重。 |

## 五、推广数据日报（专家增强版）

【发财】（CEO）要求【如一】（分析）在每日 23:00 产出的报表中必须包含 **「流量结构比」**：

```text

----------------------------------
1. 财务指标
- 总花费: ￥{XXX} | 预估毛利: ￥{XXX} | 推广成本占比: {X}%
- 整体 ROI: {X.X} (目标: > 2.0)

2. 流量分布
- 付费流量 UV: {XXX} (占比 {X}%) | PPC: ￥{0.XX}
- 免费搜索 UV: {XXX} (占比 {X}%) | 推荐 UV: {XXX}

3. 推广漏斗
- 推广中商品: {N} 款 | 产生收藏加购商品: {N} 款

- 建议: @美含 需针对加购前三的商品在明日 12:00 前完成详情页「限时限量」文案加固。
```

## 六、推广循环

推广与选品上架并行，形成持续循环：

```text
选品上架（蓝海指数校验）→ 攒满 300 款 → 高潜力款开车 → 按周三级淘汰
     ↑                                                          ↓
     └──── 持续并行选品上架 ←──── 潜力款优化 / 无效计划腾出预算给新品
```

---

## 七、架构建议（蓝海词店专项）

1. **【如一】的选词工具化**：建议给【如一】配置调用「生意参谋」或「第三方蓝海挖掘工具」的接口，使其能自动发现蓝海词。
2. **【美含】的文案差异化**：蓝海产品往往因主流市场未满足需求而存在，美含的文案不能「大众化」，要针对**细分痛点**（如：大码、特定场景、特殊功能）猛打。
3. **【发财】的预算策略**：在蓝海模式下，若【如一】发现某批词 ROI 极高且稳定，【发财】应具备「自动追加 20% 预算」的权限，无需等老板手动确认。
